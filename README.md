# Todo App

![screenshot](images/screenshot.jpg)

## General Info

Simple to-do list app built with Vanilla JS and fetching data from the Bored API.

Click [here](https://marekjanik.github.io/todo-app/dist/index.html) to watch it live.

## Technologies

JavaScript
Sass
gulp/webpack
